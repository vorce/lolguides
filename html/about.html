ABOUT

Lolguides started out as a personal little resource, to help me quickly access only the most reliable and up to date champion guides.

At the moment only solomid's and clgaming's guides are included. Later I might add leaguecraft, mobafire or other places with quality LoL guides.

The motivation for this site was simply that it was starting to get frustrating having to check multiple sites for guides. Now the best and latest are in one place! Hoorah.

Some technical details for the curious: The data is scraped by a python script (using BeautifulSoup) and output to a nice, clean JSON dump. Another python program is responsible for generating all the champion pages (and the index, but not this about page) from the data dump. The site is rebuilt a whole bunch of times per day to make sure it stays up to date.
